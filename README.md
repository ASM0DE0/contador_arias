# Contabilidad & Soluciones ARIAS

Sitio web profesional para Contabilidad & Soluciones ARIAS - Felipe Arias, Contador Auditor.

## 🚀 Tecnologías

- **React 18+** - Framework UI
- **TypeScript** - Tipado estático
- **Vite** - Build tool y dev server
- **Tailwind CSS** - Framework CSS utility-first
- **Lucide React** - Iconos modernos

## 📦 Instalación

```bash
npm install
```

## 🛠️ Desarrollo

```bash
npm run dev
```

El servidor de desarrollo se iniciará en `http://localhost:5173`

## 🏗️ Build para Producción

```bash
npm run build
```

Los archivos optimizados se generarán en la carpeta `dist`

## 📋 Linting

```bash
npm run lint
```

## 🚀 Despliegue en Vercel

Este proyecto está configurado para desplegarse directamente en Vercel sin modificaciones adicionales.

1. Conecta tu repositorio a Vercel
2. Vercel detectará automáticamente que es un proyecto Vite
3. Haz clic en "Deploy"
